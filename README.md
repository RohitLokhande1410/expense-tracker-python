# 💰 Expense Tracker – Python CLI Application

A command-line based **Expense Tracker** built with Python to help users manage their daily spending, monitor budget, and visualize expenses effectively using Pandas and Matplotlib.

---

## 🔧 Features

✅ Add new expenses with name, amount, category  
✅ View all recorded expenses in a table format  
✅ Filter expenses by date range  
✅ Delete any existing expense by name  
✅ Get monthly budget summary:
- Total spent
- Remaining budget
- Daily spending limit  
✅ Visualize expenses via pie chart (category-wise)  
✅ Error logging using Python's built-in `logging` module  

---

## 📌 Technologies Used

- Python 🐍  
- Pandas  
- Matplotlib  
- CSV for storage  
- Logging  
- OOP (Object-Oriented Programming)
